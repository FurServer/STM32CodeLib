芝士泡泡学STM32时写的一些以后可能会用到的库
---

**库列表:**

AHT20

> 单纯的发送读取, 以及单纯的读取温湿度
>

LCD1602

> LCD1602 (HD44780) 驱动, 带有类u8g2移植层, 有一大坨可用API

MPU6050

> 精简的MPU6050库, 解算出四元数和姿态角

printf

> 以便将printf重定向至串口

Switch_I2C1

> STM32F103系列48pin芯片设计缺陷, 当使能I2C1时, PB5只能作为GPIO使用
>

u8g2_stm32

> [u8g2](https://github.com/olikraus/u8g2)的硬件接口
